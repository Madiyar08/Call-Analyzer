# Отчёт операторов по звонкам

## Описание проекта
Streamlit приложение для автоматизации ежедневных отчётов операторов колл-центра. Позволяет загружать .xls/.xlsx файлы от менеджера и автоматически формировать отчёт с подсчётом потерянных и принятых звонков по часам.

## Основные функции
- Загрузка Excel файлов (.xls, .xlsx)
- Автоматический подсчёт звонков по 24 часовым интервалам (00:00-01:00 до 23:00-00:00)
- Подсчёт количества операторов в каждый час
- Расчёт итогов
- Экспорт в Google Sheets
- Скачивание готового отчёта в Excel формате

## Структура проекта
```
├── app.py                  # Основное Streamlit приложение
├── .streamlit/
│   └── config.toml         # Конфигурация Streamlit
├── attached_assets/        # Прикреплённые файлы
└── replit.md              # Документация проекта
```

## Зависимости
- streamlit - веб-интерфейс
- pandas - обработка данных
- xlrd - чтение .xls файлов
- openpyxl - чтение .xlsx файлов
- xlsxwriter - создание Excel файлов
- gspread - интеграция с Google Sheets

## Интеграции
- Google Sheets - для экспорта отчётов

## Запуск
```bash
streamlit run app.py --server.port 5000
```

## Формат отчёта
Таблица содержит:
- Столбец "Время" с интервалами (00:00-01:00, 01:00-02:00, ... 23:00-00:00)
- Для каждой даты три столбца: Потерянные, Принятые, Оператор
- Строка "Итог" с суммами
